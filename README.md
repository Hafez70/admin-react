# Taski Admin Panel

A modern, Persian (RTL) admin dashboard built with React 19, Material-UI v7, and Vite.

## 🚀 Quick Start

### Prerequisites
- Node.js 22+ installed
- Bun package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/Hafez70/taski-admin.git
cd taski-admin

# Install dependencies
bun install

# Run development server
bun run start
```

The app will open at `http://localhost:5173`

### Available Commands

```bash
bun run start    # Start development server
bun run build    # Build for production
bun run preview  # Preview production build
bun run lint     # Run ESLint
bun run lint:fix # Fix ESLint errors
bun run prettier # Format code with Prettier
```

- ✅ React 19 with functional components
- ✅ Material-UI v7 with custom theming
- ✅ RTL (Right-to-Left) support for Persian
